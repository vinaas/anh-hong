Hướng dẫn cài đặt SPM Accounting

- Trên máy chủ:

bước 0. Cài đặt .netframework 3.5 cho window xp hoặc thấp hơn (window 7 trở lên không cần cài đặt)

bước 1. Cài đặt sql server 2008 r2 sp 2 (hoặc cao hơn). SQL Server 2008 R2 Express with Tools,SQLEXPRWT, Link https://www.microsoft.com/en-us/download/details.aspx?id=30438

bước 2. Cài đặt stp service 6.1 trở lên

chú ý:
  • Nếu máy đã có stp service bản nhỏ hơn thì phải remove trước
  • Thư mục cài đặt phải để ở ổ khác ổ chứa window
  • Sau khi cài đặt xong mở chương trình "sql server connection" để khai báo thông tin kết nối với SQL SERVER

- Trên máy trạm:

Click vào file setup.exe để cài đặt
Chú ý: Nếu là window 7 hoặc 8, 8.1: Thư mục cài đặt phải để ở ổ khác ổ chứa window. nếu cài đặt cùng ổ địa với ổ chứa window thì phải phân quyền cho thư mục cài đặt chương trình có khả năng đọc và ghi dữ liệu
